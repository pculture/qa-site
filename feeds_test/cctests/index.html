feed test 1

RSS 2.0 feed with one item. Item has no Creative Commons license metadata. The item in this feed will have: 

   License: see permalink

in the item details.

  • feedtest1.xml

feed test 2

RSS 2.0 feed with one item. Item has Creative Commons license metadata. The item in this feed will have: 

   License: Attribution-Noncommercial-No Derivative Works 2.0 Generic

in the item details. And clicking on that link will open up a browser and go to the page at http://creativecommons.org/licenses/by-nc-nd/2.0/ .

  • feedtest2.xml

feed test 3

RSS 2.0 feed with one item. Item has a blip license, but no Creative Commons license metadata. The item in this feed will have: 

   License: see permalink

in the item details.

Some future version of Miro should discriminate between the two different license tags as they're in different namespaces. So this particular situation is a bug.

  • feedtest3.xml
